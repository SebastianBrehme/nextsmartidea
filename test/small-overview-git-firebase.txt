https://firebase.google.com/docs/cli/

1. Node.js download and install (latest Version v.6.8.0)
2. open cmd.exe
3. npm install -g firebase-tools
4. firebase login 
5. Go to a location
6. firbase init
7. select Hosting...
8. ...follow steps...
9. firbase deploy
